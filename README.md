2. M2M 통신 방식 중 Cellular 2G/3G/4G, WIFI, IP 네트워크 등 다양한 유형을 연구
3. 무선채널 상에서 멀티미디어(동영상), Voice, Audio 데이터 전송 시 성능 향상에 대한 연구
4. 무선채널에서의 Concatenated coding과 Hybrid ARQ 연구
5. 자기공명 방식의 무선전력시스템 관련 RLC 회로 개발
  

## 연구 경력
